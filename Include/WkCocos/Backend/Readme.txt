This folder contains all classes to use the different backends in WkCocos :
- Soomla http://soom.la/ - Used for Shop only now ( but Profile and Levelup available ... )
- App42 http://api.shephertz.com/ - Used for all online game features ( but other backends can be chosen ) 
- Device http://developer.android.com/index.html or iOS - Used for all things that depends on Device directly ( without Cocos2dx help )
- Cocos2dx http://www.cocos2d-x.org/ - Used for all features that are tied to cocos2dx.
- StaticWeb - Used for DLC download at the moment ( but can be enhanced to handle all requests to custom static website )
